# Content Factory — The Growth Operating System

> One input. Seven platforms. Revenue you can track.

Content Factory is a premium SaaS platform ($900/month) that transforms a single content idea into optimized, platform-specific content for 7 channels — powered by a sophisticated AI engine that learns your brand voice, understands your competitive positioning, and generates content with tactical depth.

## What It Does

**Turn your expertise into a 24/7 inbound pipeline machine.**

Content Factory replaces the $4,500/month you'd spend on freelancers, agencies, and disconnected tools with a single intelligent system that:

1. **Learns your brand** — Through an AI-powered interview, smart questions, or content analysis, the system builds a complete Brand Intelligence Profile.
2. **Generates 7x content from 1 idea** — Enter a topic and get optimized content for Twitter/X, LinkedIn, Instagram, Email Newsletter, Blog/SEO, YouTube, and Video Scripts.
3. **Uses 12 proven frameworks** — Contrarian + Proof, PAS, Story-Lesson-Action, Before/After Bridge, Data-Insight-Application, and more.
4. **Scores every piece** — Quality Controller evaluates Hook Strength, Specificity, Tactical Depth, Voice Match, CTA Clarity, and Platform Optimization.
5. **Generates strategic topic ideas** — AI-powered Topic Generator creates 10 buyer-journey-mapped content ideas.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **AI**: OpenAI GPT-4.1-mini
- **Deployment**: Vercel

## Getting Started

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

## Environment Variables

```env
OPENAI_API_KEY=your-openai-api-key
OPENAI_BASE_URL=https://api.openai.com/v1
```
